# 5. Sistema de captura oficial

No existe recurso de captura externo.

La captura ocurre directamente al ganar combate.

## Tabla de captura

  Rareza     Captura directa   Fragmentos
  -------- ----------------- ------------
  Común                  60%            5
  Raro                   40%           15
  Épico                   5%           25

## Regla de fallo

Si falla la captura: - El jugador recibe fragmentos de esa especie.

Ejemplo: - Fragmento SlimeFuego - Fragmento Titanox
